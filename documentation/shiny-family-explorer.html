General Conference semantic trends

Family-friendly explainer · Personal research demo · Not affiliated with the Church

What this is

A computer looks at wording patterns in every General Conference talk (1971–2021) in a public corpus, using an embedding model (BAAI/bge-small-en-v1.5). We compare each chunk of text to hand-picked example sentences for a prescriptive pole vs an invitational pole, then average to a talk-level score. This is language geometry, not theology or church policy.

Interactive app (R Shiny)

Clone or download the conference-stats repo, install R packages listed in analysis/shiny_gc_family/HOW_TO_RUN.md, then from the repo root: 

shiny::runApp("analysis/shiny_gc_family", launch.browser = TRUE)

The app works offline once bundled: plots in www/, tables in data/. Rebuild plots and data with Rscript analysis/plot_gc_chunk_embed_results.R.

Chunk insights tab

When chunk_highlights.rds is present, you can read the passages that were highest / lowest net score within each talk (not necessarily “prescriptive” on an absolute scale), plus high-leverage segments. Short “amen” closings and the last chunk of each talk are usually filtered out.
